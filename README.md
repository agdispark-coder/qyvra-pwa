# QYVRA - Women's Wellness PWA

Track your cycle, nutrition, and feel your best every day.

## About

QYVRA is a Progressive Web App (PWA) built with Flutter for women's wellness tracking. It provides cycle tracking, nutrition monitoring, and personalized health insights.

## Tech Stack

- **Flutter 3.19.0** (Web)
- **Provider** for state management
- **fl_chart** for data visualization
- **Google Fonts** (Poppins)

## Features

- Cycle phase tracking with calendar view
- Daily nutrition & calorie monitoring
- Macronutrient breakdown (Carbs, Protein, Fat)
- Meal logging
- Weekly calorie charts
- Dark mode support
- PWA: Install on any device

## Deployment

This project deploys automatically via GitHub Actions to Netlify.

### Setup

1. Push code to `main` branch
2. GitHub Actions builds Flutter web
3. Deploys to Netlify automatically
4. Visit Netlify URL and use PWABuilder to generate APK
